# Project Information

This project is a "companion app" for the SHL with a custom backend (https://api.lrlnet.se/). This is an Apple native app for iOS and iPad and we want to integrate well into the Apple ecosystem.

## Designing and Previewing

When designing new components or just changing components. We need to preview them using the Xcode MCP (if not available, ask user to enable it) in all variants (iPad/iOS, dark/light, bigger/smaller devices), to check for bad contrasting colors, weird layouting, clipped content and other weird bugs.
